THEME INSTALLATION

1)Move the Mada theme folder inside your moodle theme folder

2)Log in on your moodle as administrator and click on notifications to install your moodle

3)Click on appearance to select Mada theme.


Now you can enjoy the design of Mada theme



This theme is has a clean design and can support Moodle 2.3 version.
